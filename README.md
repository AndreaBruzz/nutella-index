# Nutella Index

Nutella Index is a Next.js application for tracking and comparing Nutella prices worldwide.

## Main Features

- Interactive world map of observations
- Country ranking with detailed entry tables
- User submission flow with moderation-ready data model
- Localized UI (`it`, `en`)

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts

```bash
npm run dev
npm run lint
npx tsc --noEmit
```
